# Sprint 0 - Setup inicial

## Escopo

- Next.js com App Router, TypeScript e Tailwind CSS.
- Estrutura base para rotas, componentes, lib, tipos, Supabase e docs.
- shadcn/ui configurado com componentes iniciais criados em `components/ui`.
- Layout inicial com sidebar desktop, navegação mobile, header e placeholders.
- Variáveis de ambiente documentadas em `.env.example`.

## Observação de ambiente

Esta máquina não possui `node`, `npm` ou `npx` no PATH da sessão atual. Por isso o scaffold foi criado manualmente. Em um ambiente com Node.js instalado, os scripts padrão devem funcionar.

## Validação esperada

```powershell
npm install
npm run dev
npm run lint
npm run build
```
